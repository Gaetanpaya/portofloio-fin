Projet 1 : Déploiement d'un Proxmox Backup Server (PBS)

Cadre : Projet de spécialité SISR (Épreuve E4)

Contexte et Objectifs

Dans le cadre de la sécurisation d'une infrastructure virtualisée sous Proxmox VE, l'objectif de ce projet était de mettre en place une solution de sauvegarde centralisée et optimisée. J'ai choisi de déployer Proxmox Backup Server (PBS) pour garantir la sauvegarde et la restauration rapide des machines virtuelles (VM) et des conteneurs (CT).


Environnement Technique

  • Hyperviseur : Proxmox Virtual Environment (PVE)
  • Serveur de sauvegarde : Proxmox Backup Server (PBS)
  • Réseau : Configuration IP statique, liaison réseau dédiée aux sauvegardes.

Étapes de réalisation

  • Installation : Déploiement de l'OS Proxmox Backup Server sur une machine dédiée (ou en VM isolée).
  • Configuration du stockage : Création et formatage des Datastores (ZFS/Ext4) pour accueillir les sauvegardes.
  • Liaison PVE / PBS : Ajout du serveur PBS comme espace de stockage sur le cluster Proxmox VE via l'échange de clés d'authentification.
  • Automatisation : Création de tâches planifiées (Backup Jobs) pour sauvegarder les VM toutes les nuits.
  • Gestion de la rétention : Configuration des règles de nettoyage (Prune) pour ne garder que les X dernières sauvegardes et optimiser l'espace disque (déduplication).
  • Tests : Simulation d'un crash et restauration réussie d'une VM complète.

Compétences SIO mobilisées

  • Mettre en place une solution de sauvegarde et de restauration.
  • Administrer des systèmes (Linux, Proxmox).
  • Gérer le patrimoine informatique.
